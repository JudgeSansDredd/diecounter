# Die Counter

## Purpose

Sometimes, it's hard to count up your die pool. We aim to fix that problem.

## Local Development

1. Create a virtual environment
   ```
   python -m venv venv
   ```
2. Activate virtual environment
   ```
   ./venv/bin/activate
   ```
3. Install requirements
   ```
   pip install -r requirements.txt
   ```
4. Run `main.py`
   ```
   python main.py
   ```
